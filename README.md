# java-cli-maven-h2-simple

## Description
A java maven build, that connects to h2
database.

H2 is a database written in `java`
and is best known for a `dev` only
use.

## Tech stack
- java
- maven
- h2 driver

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
